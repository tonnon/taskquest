# Taskquest

Aplicação de produtividade gamificada construída com Vite + React.

## Desenvolvimento local

Pré-requisitos:

- Node.js 18+
- npm

Passos para rodar:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Faça o build com:

```sh
npm run build
```

Em seguida publique o conteúdo de `dist/` na sua plataforma de hospedagem preferida (Vercel, Netlify, etc).
